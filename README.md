# Restaurant Data Analysis Project

This project was completed as part of the CodeAlpha Data Analytics Internship.

I used Python to perform Exploratory Data Analysis (EDA) on a restaurant dataset containing information about restaurant ratings, customer votes, cuisines, pricing, and delivery services.

## What I Analyzed

- Dataset structure and dimensions
- Missing values
- Duplicate records
- Restaurant distribution across cities
- Customer ratings
- Popular restaurants based on votes
- Most common cuisines
- Average cost analysis
- Correlation between numerical features

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Files Included

- CodeAlpha_Task2_EDA.ipynb
- Dataset.csv

## Key Observations

- New Delhi contains the highest number of restaurants in the dataset.
- Restaurant ratings vary significantly across locations.
- Some restaurants receive substantially higher customer votes than others.
- Customer ratings and votes provide useful insights into restaurant popularity.

## Conclusion

This analysis helped me understand how to explore, clean, and interpret real-world datasets using Python.
